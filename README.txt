EasyBounce – Installation Guide
────────────────────────────────────────────────────────────


1. Install the app
   Drag EasyBounce.app into your /Applications folder.


2. Apply the one-time launch patch
   Open Terminal (Finder → Applications → Utilities → Terminal)
   and run the following command:

       xattr -cr /Applications/EasyBounce.app


3. Grant Accessibility access
   System Settings → Privacy & Security → Accessibility
   → Enable EasyBounce

   ⚠️  If EasyBounce doesn't appear or the toggle doesn't stick:
   Remove it from the list using the − button,
   then add it back using the + button.


4. Grant Automation access
   System Settings → Privacy & Security → Automation
   → Enable all three items under EasyBounce:
      • Logic Pro
      • System Events
      • Finder


5. You're all set
   Open EasyBounce from your /Applications folder.
   EasyBounce opens automatically once all permissions are granted.


────────────────────────────────────────────────────────────
Note: Steps 3–4 only appear on first launch.
EasyBounce guides you through them automatically.
