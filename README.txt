EasyBounce
──────────────────────────────────────────

1. Drag EasyBounce into your Applications folder.

2. Open EasyBounce.
   macOS will ask for Accessibility permission —
   click OK and enable the toggle.

   ⚠  If the toggle won't stay on:
   remove EasyBounce from the list using the  −  button
   then add it back using the  +  button.

3. You're all set — enjoy!

──────────────────────────────────────────
System Settings → Privacy & Security → Accessibility
